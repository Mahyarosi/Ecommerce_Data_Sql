# Zepto E-Commerce Data Analysis

This project demonstrates data cleaning, transformation, and analysis of an e-commerce product dataset using SQL Server.

## Project Overview
- Checked for missing/null values
- Explored product categories and stock status
- Cleaned data: removed invalid prices and converted paise to rupees
- Performed analysis queries:
  - Top 10 best value products
  - High MRP out-of-stock products
  - Estimated revenue per category
  - Products with MRP > 500 and low discounts
  - Top categories by average discount
  - Price per gram for products above 100g
  - Categorized products by weight
  - Total inventory weight per category

## How to Run
1. Open SQL Server Management Studio (SSMS)
2. Load the SQL scripts in this repository
3. Execute queries in order to reproduce results
4. Optional: export results as `.csv` for reports or visualization
